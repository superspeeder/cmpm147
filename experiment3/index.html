Experiment 3 - Alternate Worlds

Dungeon Generator

Description

This is a simple dungeon generator which generates room layouts using binary spatial partitioning.


Codes

The code _ is the in the wall code (tiles not in the open space of the room or on its boundary). The # is the room/wall boundary code, which was used to make it easier to autotile the walls. The . tile is the open air tile code.


Life

When you move your mouse around the scene, the mouse is a simulated light source of sorts, which uses raycasting and a transparent shape to fake lighting.
